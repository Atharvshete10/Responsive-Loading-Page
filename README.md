# PRODIGY_WD_01 - Responsive Landing Page

## 📌 Task Description

Create a responsive landing page with an interactive navigation menu. The navigation menu changes its style when the user scrolls the page or hovers over menu items. The navigation bar remains fixed at the top and is visible on all sections of the page.

## 🚀 Features

- Responsive Landing Page
- Fixed Navigation Bar
- Navigation Bar Changes on Scroll
- Hover Effects on Navigation Links
- Smooth Scrolling
- Modern UI Design
- Mobile-Friendly Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
